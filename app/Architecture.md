
## 整体流程
```bash
                     ┌────────────────────┐
                     │      用户输入      │
                     │  （股票号码/电影名） │
                     └─────────┬────────┘
                               │
                               ▼
                    ┌────────────────────┐
                    │     输入接口       │
                    │   （RESTful API）  │
                    └─────────┬────────┘
                               │
                               ▼
                    ┌────────────────────┐
                    │     搜索模块       │
                    │  构造查询关键词    │
                    └─────────┬────────┘
                               │
                               ▼
                    ┌────────────────────┐
                    │  搜索引擎 API调用   │
                    │  （如 Bing API）   │
                    └─────────┬────────┘
                               │
                               ▼
                    ┌────────────────────┐
                    │  爬虫/抓取模块     │
                    │  获取网页HTML内容   │
                    └─────────┬────────┘
                               │
                               ▼
                    ┌────────────────────┐
                    │  文本预处理模块     │
                    │  （解析、去噪、摘要）│
                    └─────────┬────────┘
                               │
                               ▼
                    ┌────────────────────┐
                    │ 提示词组装模块      │
                    │  根据场景模板整合   │
                    │  动态生成提示词     │
                    └─────────┬────────┘
                               │
                               ▼
                    ┌────────────────────┐
                    │ 大模型 API调用模块  │
                    │ （例如 OpenAI API） │
                    └─────────┬────────┘
                               │
                               ▼
                    ┌────────────────────┐
                    │  结果返回与展示模块 │
                    │  （格式化、反馈）   │
                    └────────────────────┘
```

## 数据流程
```
   [用户输入]
        │
        ▼
  接收查询信息（股票号码/电影名）
        │
        ▼
  根据输入构造搜索关键词：
    “股票号码 东方财富 同花顺 新浪财经 搜狐财经”
    或 “电影名 豆瓣电影 百科 IMDB 冒烟”
        │
        ▼
 调用搜索引擎API，返回网页链接列表
        │
        ▼
  对返回链接逐个抓取页面HTML内容
        │
        ▼
  解析HTML，提取正文文本
        │
        ▼
   对抓取内容执行预处理：
  去除噪音、广告；若内容过长则摘要
        │
        ▼
  根据不同场景组装提示词：
   示例 —— 股票场景：
   “请给我分析【股票名】，
    参考内容如下：
    [页面1文本摘要]
    [页面2文本摘要]
    … 
    返回今天走势、明天估计、未来走势。”
        │
        ▼
 将提示词发送大模型API调用
        │
        ▼
  大模型生成分析/推荐结果
        │
        ▼
  将结果进行解析、格式化后展示给用户

```
